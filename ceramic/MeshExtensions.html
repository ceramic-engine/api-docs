Ceramic API

class MeshExtensions

package ceramic

Available on all targets

Static extension with additional helpers for ceramic.Mesh

Static methods

@:value({ floatsPerVertex : -1 })staticcreateQuad(mesh:Mesh, width:Float, height:Float, floatsPerVertex:Int = -1):Void

Generate vertices, indices and uvs on the given mesh to make it form a quad

Parameters:

mesh

The mesh to work with

width

With of the quad to form

height

Height of the quad to form

floatsPerVertex

(optional) Number of floats per vertex Set to 2 for regular quad, 6 for quads with dark color, must be 2 or higher. If not provided, will resolve the value from mesh.customFloatAttributesSize

staticsetDarkColor(mesh:Mesh, darkColor:Color):Void

Assign a dark color to the given mesh. The mesh is expected to have 6 floats per vertex and 4 last vertices will be used for color values.

Parameters:

mesh

The mesh to work with

darkColor

The dark color to assign

staticsetDarkAlphaColor(mesh:Mesh, darkAlphaColor:AlphaColor):Void

Assign a dark color (with alpha included) to the given mesh. The mesh is expected to have 6 floats per vertex and 4 last vertices will be used for color values.

Parameters:

mesh

The mesh to work with

darkColor

The dark color to assign

staticcreateArc(mesh:Mesh, radius:Float, angle:Float, thickness:Float, sides:Int, borderPosition:BorderPosition):Void

Generate vertices to draw arc, pie, ring or disc geometry

Parameters:

mesh

The mesh to work with

radius

Radius of the arc

angle

Angle (from 0 to 360). 360 will make it draw a full circle/ring

thickness

Thickness of the arc. If same value as radius and borderPosition is INSIDE, will draw a pie.

sides

Number of sides. Higher is smoother but needs more vertices

borderPosition

Position of the drawn border
